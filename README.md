# 🩺 Doctor Portfolio Website

A modern, responsive **doctor portfolio website** designed for medical professionals.
Built using **HTML, CSS, and JavaScript**, this project showcases a professional doctor profile with services, medical expertise, and a patient contact form.

## ✨ Features

🩺 **Professional Hero Section**
A clean landing section introducing the doctor and specialization.

📊 **Experience & Achievements**
Highlights years of experience, patient success rate, and medical milestones.

🏥 **Medical Expertise Section**
Showcases specialization areas such as cardiology, diagnostics, and patient care.

🧠 **Advanced Healthcare Approach**
Focus on AI-driven diagnostics and preventive healthcare.

📬 **Patient Contact Form**
Allows patients to send consultation requests easily.

📱 **Fully Responsive Design**
Works smoothly on **mobile, tablet, and desktop devices**.

🎨 **Modern UI Design**
Minimalist interface with smooth animations and professional styling.

## 🛠️ Technologies Used

* HTML5

* Responsive Web Design
